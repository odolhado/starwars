

# Star Wars - Code&Pepper

 * NodeJS
 * NestJS
 * Swagger
 * rest API
 * hexagonal architecture


## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Open Swagger

http://local:3000/api/#/characters

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
